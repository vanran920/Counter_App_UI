## Counter App 

##  Basic Feature :- 

  +  -> Increase Value
  -  -> Decrease Value till it reaches zero
  Add +5 -> add 5 value
  Sub -5 -> delete 5 value
  Reset -> It reset the value

  Value will never go below zero 

  You can input any value you want , but only integers.
